# 🧩 Система плагинов AlaquBot

## Обзор

AlaquBot использует **плагинную архитектуру**, которая позволяет добавлять новые функции без изменения ядра бота.

## Основные принципы

| Принцип | Описание |
|---------|----------|
| **Авто-обнаружение** | Бот сам находит плагины в папке `src/plugins/` |
| **Изоляция** | Каждый плагин работает независимо |
| **Конфигурация** | Настройки через YAML файлы |
| **Версионирование** | Каждый плагин имеет версию |

## Как это работает

```
1. Бот запускается
   ↓
2. Сканирует директорию src/plugins/
   ↓
3. Для каждой директории загружает plugin.py
   ↓
4. Читает конфиг из src/plugins/<name>/config.yaml
   ↓
5. Регистрирует команды
   ↓
6. Плагин готов к работе
```

## Преимущества

- ✅ Добавляй новые команды без изменения ядра
- ✅ Настраивай поведение через YAML
- ✅ Отключай плагины одной строкой
- ✅ Делись плагинами с сообществом
