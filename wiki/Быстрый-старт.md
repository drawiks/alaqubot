# 🚀 Быстрый старт

Создадим простой плагин "hello" за 5 минут.

## Шаг 1: Создай папку плагина

```
src/plugins/hello/
├── __init__.py
└── plugin.py
```

## Шаг 2: Создай __init__.py

```python
from .plugin import HelloPlugin

__all__ = ["HelloPlugin"]
```

## Шаг 3: Создай plugin.py

```python
from src.core.plugin import Plugin


class HelloPlugin(Plugin):
    name = "hello"
    version = "1.0.0"

    async def on_load(self) -> None:
        """Вызывается при загрузке плагина"""
        pass

    async def on_unload(self) -> None:
        """Вызывается при выгрузке плагина"""
        pass

    def get_commands(self) -> list[dict]:
        return [
            {"name": "привет", "handler": self.say_hello, "config": self.get_command_config("привет")},
        ]

    async def say_hello(self, cmd):
        await cmd.reply(f"Привет, {cmd.user.name}!")
```

## Шаг 4: Создай конфиг

```
src/plugins/hello/
├── __init__.py
├── plugin.py
└── config.yaml
```

```yaml
enabled: true

commands:
  привет:
    cooldown: 0
    enabled: true
```

## Шаг 5: Перезапусти бота

Плагин загрузится автоматически!

## Результат

Теперь команда `!привет` будет работать:

```
Пользователь: !привет
Бот: Привет, Username!
```
