# 📂 Структура плагина

## Обязательные файлы

Каждый плагин должен содержать:

```
src/plugins/<имя_плагина>/
├── __init__.py      # Экспорт плагина
├── plugin.py        # Код плагина
└── config.yaml      # Конфигурация
```

## Описание файлов

### __init__.py

Точка входа плагина. Просто экспортирует класс плагина.

```python
from .plugin import MyPlugin

__all__ = ["MyPlugin"]
```

### plugin.py

Основной файл плагина. Содержит класс, наследующий `Plugin`.

```python
from src.core.plugin import Plugin


class MyPlugin(Plugin):
    name = "myplugin"
    version = "1.0.0"

    # Обязательные методы
    async def on_load(self): ...
    async def on_unload(self): ...
    def get_commands(self): ...

    # Твои команды
    async def my_command(self, cmd): ...
```

### config.yaml

Конфигурация плагина в формате YAML.

```yaml
enabled: true

commands:
  команда1:
    cooldown: 10
    enabled: true
  команда2:
    cooldown: 0
    enabled: true
    requires_permission: true

settings:
  # Кастомные настройки
  my_option: value
```

## Именование

| Элемент            | Правило | Пример |
|--------------------|---------|--------|
| Директория плагина | lowercase | `hello_world/` |
| Класс плагина      | PascalCase | `HelloWorldPlugin` |
| Имя в конфиге      | lowercase | `hello_world` |

## Полный пример

```
src/plugins/hello/
├── __init__.py
├── plugin.py
└── config.yaml
```
